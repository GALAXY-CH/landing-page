# Galaxy Switzerland Landing Page

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

A modern, responsive landing page for Galaxy Switzerland - a powerful, web-based platform for accessible, reproducible, and transparent computational research in life sciences.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional interface with smooth animations
- **Accessible**: Built with accessibility best practices
- **Fast Loading**: Optimized performance and asset delivery
- **SEO Optimized**: Comprehensive meta tags and semantic HTML

## Project Structure

- **Navigation**: Quick links and Galaxy launch button
- **Hero**: Eye-catching introduction with call-to-action
- **About**: Overview of Galaxy Switzerland features and statistics
- **Features**: Grid showcasing key platform capabilities
- **Community**: Links to support, training, and events
- **Footer**: Resources and Galaxy network links

## Technologies

- HTML5
- CSS3
- JavaScript (Vanilla)
- Google Fonts (Inter)

## Getting Started

1. Clone or download the repository
2. Open `index.html` in your web browser
3. All assets should be in the `assets/` folder
